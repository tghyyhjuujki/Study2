### 보안 CIA

C (Confidentiality) : 기밀성, 데이터를 인가된 사람에게만 허용

I (Integrity) : 무결성, 데이터가 변조되지 않는 것

A (Availability) : 가용성,  둘을 충족시키면서 사람들이 데이터를 이용가능하게 함

<br/>

### 인가, 인증

인증 : 시스템에 등록된 사용자인지 여부를 확인하는 것

인가 : 인증된 사용자에게 권한을 부여함
